Змінні та типи даних

1. Чим відрізняються змінні const та let?
   а) змінні const не можна змінювати після їхньої ініціалізації, а змінні let можна
   б) змінні let не можна змінювати після їхньої ініціалізації, а змінні const можна
   в) нічим не відрізняються

2. JavaScript є динамічною мовою програмування. Що це означає?
   а) у ній є декілька типів даних
   б) типи даних визначаються автоматично в процесі виконання програми
   в) тип даних має бути визначений явно

3. Що виведе такий код: typeof null?
   а) "null"
   б) "string"
   в) "object"

4. Яке твердження про рядки (strings) у JavaScript є правдивим?
   а) рядки можуть буди включені ЛИШЕ в подвійні лапки ("")
   б) рядки не можна змінювати
   в) розривати рядки не можна

5. Як отримати довжину рядка?
   а) string.length
   б) string.length()
   в) string.len
